# Heart_Disease_Project
Machine learning analysis of the UCI Heart Disease dataset using classification, regression, clustering, model optimization, and scenario analysis. Includes SVM, XGBoost, K-Means with PCA, sensitivity analysis, and business insights developed as part of an ECPI University AI/ML project.
# Heart Disease Prediction Project

## Overview

This project applies machine learning techniques to the UCI Heart Disease dataset to analyze factors associated with cardiovascular disease. The project includes classification, regression, and clustering tasks, followed by model optimization and scenario analysis.

## Dataset

* UCI Heart Disease Dataset
* Features include age, cholesterol, blood pressure, heart rate, chest pain type, and other clinical measurements.

## Analytical Tasks

### Classification

An optimized Support Vector Machine (SVM) classifier was developed to predict the presence of heart disease.

### Regression

An XGBoost regression model was implemented to explore relationships between selected clinical variables.

### Clustering

K-Means clustering with PCA was used to identify groups of patients with similar characteristics.

## Scenario Analysis

Three realistic healthcare scenarios were evaluated:

1. Improved Cardiovascular Health
2. Increased Cardiovascular Risk
3. Aging Population

The optimized SVM model was used to predict the impact of each scenario on heart disease risk.

## Sensitivity Analysis

A sensitivity analysis was performed by modifying cholesterol and resting blood pressure values. The results demonstrated how changes in these features affected model predictions.
